# infEGE
Все задачи ЕГЭ распределены по папкам с соответствующими номерами.\
Папка конкретного задания включает в себя 2 папки: с шаблонами и домашними задачами.\
Ученик добавляет в папку "hw" все решенные дома задачи, используя в качестве шаблона названия файла/папки: *"01_09_2025_1234"*, где 1234 - номер задания в кодификаторе сайта полякова.\
В случае, когда задач в одном домашнем задании больше 1 на конкретную дату создается отдельная папка.
